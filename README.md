# calculadora financiera app
Aplicación web que permite calcular cuotas de préstamos con interés.

# Demo
https://abomdevp.github.io/calculadora-financiera-app/

# Funcionalidades
- Cálculo automático de cuota mensual
- Validación de inputs con mensajes claros
- Diseño responsive (mobile first)
- Tema claro/oscuro
- Validación de formularios
- Accesibilidad WCAG 2.1 AA

# Uso
1. Clonar o descargar el proyecto
2. Abrir index.html en el navegador
3. Listo (no requiere instalar dependencias)
